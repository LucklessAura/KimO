Problem Description

About

KimO is a website that aims to help in child supervision

Tehnologies used

  • Php
  • Html 5
  • Css
  • Sql

Description

KimO will use 'Some beacon API' to get your current location, as a supervisor, as well as the location of children that you added throug the Child Details page, the webpage will then show you using 'Some map API' on a live map your location, your children's location as well as other users and their respective childern that are in the vicinity. The webpage will also allert you about different dangers, the type of allert will be chosen by the user. The webpage will also offer information about the distance between you and the children and allert if a certain range is exceeded. At the end of the day in the Child Details page you will be able to see the activity of your child during the day.

What needs to be implemented

  • Database to remember users and their children
  • Choosing a beacon API that can provide geographical coordonates
  • Choosing a map API that can show locations based on geographical coordonates
  • Creating a transparent overlay over the map where different points can be displayed
  • Creating function that gets coordonates from beacon and sends them to the map
  • Create function that gets pint from map and shows them on overlay
  • Create function that can calculate distance between two beacon locations
  • Function that record time spent near another beacon if distance between them is small
  • Functions that can send messages, play sounds, send emails, etc.
  • Fuction that recieves impulses and prints a message on screen and also calls a different function that will proceed to take a different action
  • Function for login

Data Aquisition

  • Data for database from forms in the webpage
  • Location coordonates from a beacon API
  • Distances from operations aplicated to the coordinates of the beacons
  • Impulses either manually from a console or trough a script that sends them tot the apropriate function
